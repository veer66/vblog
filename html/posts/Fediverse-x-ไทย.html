Fediverse x ไทย

Tagged as fediverse
Written on 2021-06-04 00:27:04

Fediverse เป็น social network ที่ประกอบไปด้วย server ย่อย ๆ ที่เป็นของคนละเจ้าของกัน มีแอดคนละคน มีกติกาต่างกัน แต่ว่าคุยกันข้าม server ได้ถ้าไม่โดน block

เดิมทีผมรู้จักคนที่ไทยที่ใช้ Fediverse ไม่เกินสองคน แต่ผมก็คิดว่าไม่เป็นไรผมใช้ server ต่างประเทศคุยภาษาอังกฤษก็ได้

แต่พอลองมองย้อนไปแต่ก่อนก็มี Sanook ทำ Noknok ขึ้นมาเมื่อค.ศ. 2008 ผมก็ตาม @sugree ไปเล่นอยู่พักหนึ่ง แต่เขาก็เลิกไป

สองสามปีมานี้ทำให้ผมทราบว่าผมประเมินพลังคนไทยต่ำไป มี server คนไทยเกิดขึ้นมามากมายจนเกินคาด ตัวที่หลัก ๆ ที่ยืนระยะมาได้เป็นปีคือ mastodon.in.th ตอนนี้ผู้ใช้ที่ยังเข้ามาเรื่อย ๆ อยู่ร้อยกว่าท่านแล้ว ตัวนี้เป็นตัวเดียวที่ใช้ Mastodon และคนไทยจัดการ ที่เหลือก็เป็น Pleroma ได้แก่ norzeworld, pleroma.in.th และ capybara.social ส่วน ThaiTube นั้นก็กึ่ง ๆ จะ federate ดีหรือไม่ federate ดี

เดี๋ยวนี้นอกจากเรื่อง tech เรื่อง open culture แล้วเรื่องการเมืองก็ดูจะเข้มข้นยิ่งกว่า เรื่อง life style อาหารการกินก็มีทั่วไป

อนาคตรอดูอยู่ว่าจะมีใครเปิด Lemmy และ Friendica หรือไม่

server ที่เปิดมานาน ๆ จะมีการส่งต่องานกันอย่างไร ผมไม่กลัว server ปิด เพราะเจอมามากทั้ง Google Plus, Orkut, Multiply บริษัทใหญ่ก็ดีที่ว่าทุนเยอะแต่ก็ปิดอยู่ดี อาสาสมัครทุนน้อยแต่ก็ดีเรื่องการส่งต่อ ความที่รับคนน้อยค่าใช้จ่ายต่อเดือนก็ไม่ได้หนักหนาเกินกำลัง
